FreeSLW (Free Sound Library Wrapper)

FreeSLW stands for Free Sound Library Wrapper. It is written in C/C++ and the intention is that FreeSLW is to be
used in computer games, providing easy to use functions for sound playback, positioning in a 3D environment
and overall management of sound effects.

FreeSLW is free software released under the GPL version 3 license. For more info please 
read gpl.txt (that should be included with this library).
